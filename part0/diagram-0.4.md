# Exercício 0.4 - Diagrama de criação de nova nota

```mermaid
sequenceDiagram
    participant browser
    participant server

    Note right of browser: User writes note and clicks "Save"

    browser->>server: POST /new_note (note data)
    activate server
    server-->>browser: HTTP 302 Redirect
    deactivate server

    browser->>server: GET /notes
    activate server
    server-->>browser: HTML page
    deactivate server

    browser->>server: GET CSS and JS (if needed)
    server-->>browser: files

    browser->>server: GET /data.json
    server-->>browser: updated notes
